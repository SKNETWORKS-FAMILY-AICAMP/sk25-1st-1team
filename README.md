# 3. 프로젝트 개요
한국지방행정연구원(KLACI)에서 개발한 지자체 유형 분류 모델을 자동차 등록 데이터에 접목시켜 Python과 Streamlit을 활용해 구축한 데이터 기반 지자체 자동차 분포 분석. 
SQL 데이터베이스와 연동하여 대용량 자동차 등록 데이터를 효율적으로 처리하고 시각화.
## 📕 프로젝트명
KLACI 지자체 유형에 따른 자동차 분포 분석
## ✅ 프로젝트 배경 및 목적
개인의 성격(MBTI)이 행동 양식을 결정하듯, 지자체의 유형은 도시의 모빌리티 패턴을 결정짓습니다. 본 프로젝트는 화물 운송 중심의 '산업 동맥'형 도시와, 승용 이동 중심의 '생활 터전'형 도시가 갖는 구조적 차이에 주목했습니다. 이러한 도시별 기능의 차이를 정량적 데이터를 통해 입증하고 시각화했습니다.
## 🖐️ 프로젝트 소개
-지자체 성격 진단
KLACI 4대 지표(성장성, 경제성, 생활여건, 안전성)를 레이더 차트로 시각화하여 해당 지역의 강점과 약점을 한눈에 파악.
-유형별 비교 분석
선택한 지자체가 속한 유형(예: 기초안정형, 활력생활형)의 평균 데이터와 개별 데이터를 비교하여, 해당 지역만의 독특한 모빌리티 특성을 추합
-데이터 기반 스토리텔링
복잡한 수치 대신 "이 지역은 화물차 비중이 높은 산업 특화 지역입니다"와 같은 직관적인 텍스트 인사이트를 자동으로 생성하여 제공.
## ❤️ 기대효과
-지자체 교통 정책 수립: 지역 특성(화물 중심 vs 승용 중심)에 맞춘 도로 정비, 주차 공간 확보, 대중교통 노선 개편 등 맞춤형 정책 수립의 근거 자료로 활용 가능.
-데이터 리터러시 향상: 일반 시민이나 연구자가 내가 사는 도시의 객관적인 지표와 성격을 데이터로 쉽게 이해하고 비교가능
## 👤 대상 사용자
-지방자치단체 교통/도시계획 담당자: 데이터에 근거한 효율적인 교통 행정이 필요한 공무원
-도시 공학/데이터 분석 연구자: 지자체 유형과 교통 데이터 간의 상관관계를 연구하는 학생 및 연구원
